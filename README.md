# Loan Application Data Analysis (Python Practical)

## 📌 Project Overview

This project analyzes a loan application dataset using Python. It covers statistical analysis, probability concepts, data visualization, and basic linear algebra applications. The goal is to understand customer behavior and loan default patterns.

---

## 📂 Dataset Description

The dataset contains 5000 records with the following features:

* Customer_ID
* Age
* Income
* Loan_Amount
* Credit_Score
* Loan_Term (months)
* Default_Status (Yes/No)

---

## ⚙️ Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* SciPy

---

## 🔹 Step 1: Central Tendency & Dispersion

* Calculated mean, median, and mode of Income
* Calculated range, variance, and standard deviation of Loan_Amount

Purpose: To understand distribution and variability of financial data.

---

## 🔹 Step 2: Probability & Events

* Computed probability of loan default
* Created contingency table (Default_Status vs Credit Score category)
* Calculated conditional probability

Purpose: To analyze risk and likelihood of defaults.

---

## 🔹 Step 3: Distribution & Visualization

* Histogram of Credit Score with Gaussian curve
* Skewness and kurtosis of Loan Amount
* Q-Q plot for Income

Purpose: To understand data distribution and normality.

---

## 🔹 Step 4: Linear Algebra Application

* Represented customers as vectors [Income, Loan_Amount]
* Calculated dot product between vectors
* Found L2 norm (vector magnitude)
* Computed angle between vectors

Purpose: To measure similarity and relationships between customers.

---

## 📊 Key Insights

1. Customers with low credit scores are more likely to default.
2. Income distribution is fairly balanced around the mean.
3. Loan amounts vary moderately across customers.
4. High credit score customers show lower default risk.
5. Majority of customers do not default.

---

## 📌 Conclusion

This project demonstrates how statistical methods, probability, visualization, and linear algebra can be applied to real-world financial data. It helps in identifying patterns and making informed decisions in loan approval systems.
